# Contributing

First of all, thank you for considering contributing! Every contribution—whether it's a bug report, documentation improvement, feature suggestion, code contribution, performance optimization, or simply sharing an idea—is greatly appreciated.

## Ways to Contribute

You can help by:

* Reporting bugs.
* Suggesting new features or improvements.
* Improving documentation.
* Fixing bugs.
* Adding tests.
* Improving performance.
* Refactoring code.
* Reviewing pull requests.
* Sharing ideas and feedback.

No contribution is too small, and new ideas are always welcome.

## Pull Requests

For bug fixes, documentation improvements, new features, or other enhancements, feel free to open a pull request.

When submitting a pull request:

* Keep changes focused on a single purpose whenever possible.
* Write clear commit messages.
* Update documentation if your changes affect users.
* Add or update tests when appropriate.
* Make sure the project builds successfully before submitting.

Every pull request is manually reviewed before it is merged. AI-assisted contributions are welcome, but all accepted changes must meet the project's quality, security, and maintainability standards.

## Coding Style

There are no strict formatting requirements.

Please try to:

* Write clean and readable code.
* Follow the existing style of the project.
* Keep the code simple.
* Prefer maintainability over cleverness.

If possible, try to keep your code consistent with the surrounding codebase. Consistency is generally more important than personal preference.


## Security

If your contribution fixes a security vulnerability, **do not** open a public issue or pull request first.

Please follow the instructions in `SECURITY.md` for responsible disclosure.

## Code of Conduct

Be respectful, constructive, and welcoming to others. Everyone is encouraged to contribute regardless of experience level.

## Keep the Ideas Coming

Have an idea that could make the project better? Open an issue, start a discussion, or submit a pull request. Whether it's a small improvement or a big feature, I'd love to hear your thoughts.

Thank you for helping make this project better! 🚀
